## 小小練習給想找專業人才的公司參考

### 安裝環境
```
php 8.0.18
laravel 9.1.5
npm 8.7.0
```

### 安裝流程
```
composer install
cp .env.example .env => change your vars
php artisan key:gen
php artisan jetstream:install livewire
npm install && npm run dev
php artisan migrate
```
