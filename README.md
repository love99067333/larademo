### Environment
```
php 8.0.18
laravel 9.1.5
npm 8.7.0
```

### Installation
```
composer install
cp .env.example .env => change your vars
php artisan key:gen
php artisan jetstream:install livewire
npm install && npm run dev
php artisan migrate
```
